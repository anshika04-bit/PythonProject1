This Python program creates a Tkinter-based Registration System connected to a MySQL database.
It allows the user to Insert, Update, Delete, Search, and Display student registration records.
The data is shown in a TreeView table, and all operations are saved in the database.
This code builds a complete Student Registration System using Tkinter and MySQL.
It provides all CRUD operations through simple buttons and displays the data in an easy-to-read TreeView table.
The GUI includes background images, styled labels, entry boxes, dropdowns, and validation features like password visibility.
